Sync Phone to FTP server

Version 1

 

This Application aims to sync recursivly selected folders on your phone to an ftp server.

Configure FTP:

Go to Settings by pressing the dots on the right of the toolbar

If your FTP server is not accessible from the outside, you can sync when you are inside the network.

On next release, I will support sftp and ftps.

Folder Sync

After Creating the FTP, you can go to the directories you want to sync and press the left cloud button  on the toolbar so it converts to green .

By doing that, the folder is added to the files that should be synced.

Dont Forget to press the sync button  in order to start syncing the folders you selected to be synced.

Sync Rule

  • In this version, the Sync is done each time you open the application or press the sync button 
  • In this version, The backup is recursive. if you sync a folder, all its sub-folders will be synced.
  • The file will be synced only once unless you changed its name and the parent directory is still synced.
  • each time you login, the files that failed to sync will be synced again
  • If you selected a folder to be synced, The green cloud won't be green when you traverse to sub-folders.
